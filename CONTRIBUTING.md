# Contributing

This repository stores shared agent rules and skills. Do not commit private
configuration, credentials, sessions, logs, caches, company project paths, or
personal environment files.

## Change Flow

Repository administrators can commit directly to `main`.

Everyone else must change files through a pull request. The pull request must
be reviewed by an administrator before it is merged.

See `docs/git-submission-rules.md` for the normal Git submission rules.

In the pull request, explain:

- What changed.
- Why it changed.
- If `AGENTS.md` or `rules/**` changed, whether the change may conflict with
  existing rules.
