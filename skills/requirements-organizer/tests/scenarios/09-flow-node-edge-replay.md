# 场景 09：流程节点与连线回放

## 历史失败

蓝湖原型或流程图采集时，只摘录节点文字，漏掉页面连线、箭头方向、颜色分组、状态图例和低置信关系。

## 用户输入

请根据这张蓝湖总览图整理需求。图中有多个页面节点、不同颜色状态、箭头连线、部分交叉线和两个看不清方向的关联。需要能给后续开发和测试继续使用。

## 必须覆盖

- 输出节点状态矩阵，至少包含节点、状态或颜色、分组、上游、下游和说明。
- 输出连线清单，至少包含起点、终点、方向、标签或颜色、置信度和待确认。
- 对看不清方向的线标成低置信或待确认，不写成确定流程。
- 说明颜色、编号、泳道或边框样式如果没有图例，需要列为待确认。
- 把节点和连线并回需求、验收或信息缺口，不能只作为截图描述。

## 失败判定

- 只写节点，不写连线。
- 把低置信连线写成确定跳转。
- 不说明颜色、分组或图例的含义缺口。
