# Agent MCP 输出规则

- 通用证据、查询、日志、接口和工具输出先遵循 `@rules/evidence-output-rules.md`；本文件只补充 MCP 选择、调用、兜底和连接来源。
- 面向用户直接回传 MCP 查询结果时，除通用证据字段外，还要说明 MCP 工具或服务、查询目标、查询条件、返回数量和结论。
- 用户点名 MCP 或具体数据源且存在对应 MCP 时，先尝试对应 MCP；失败时说明 MCP 查询目标、失败原因和采用的只读兜底方式。
- MCP 不可用时，可以使用等价只读 API 或 CLI 兜底；最终答复必须明确这是兜底查询，不是 MCP 直接返回。
- MCP 连接信息、server 名称、资源 URI、分页/游标和端点以本机配置或本轮实际返回为准；不要在公共规则中写死地址、端口或账号。
